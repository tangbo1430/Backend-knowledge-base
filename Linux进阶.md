# Linux进阶

### 一、文件介绍

```
Linux 文件结构是一个层次化的目录结构，以根目录 / 开始，下面是对常见目录的简要分析：
/bin: 存放系统的基本命令，如ls、cp、mv等。
/boot: 存放启动 Linux 系统所需的文件，包括内核文件和启动加载程序（bootloader）。
/dev: 存放设备文件，用于与硬件设备进行交互，如磁盘、终端、打印机等。
/etc: 存放系统的配置文件，包括网络配置、用户配置、服务配置等。
/home: 存放用户的主目录，每个用户通常有一个对应的子目录，用于存放个人文件和配置。
/lib和/lib64: 存放系统的共享库文件，这些库文件被可执行文件使用。
/media和/mnt: 用于挂载可移动媒体设备（如光盘、USB 设备）和其他文件系统的挂载点。
/opt: 用于存放可选的第三方软件包，通常是一些独立安装的软件。
/proc: 虚拟文件系统，提供了对系统内核和进程的访问接口，可以通过读取其中的文件获取系统和进程的信息。
/root: root 用户的主目录，root 是系统管理员的特权用户。
/sbin: 存放系统管理命令，只有管理员或超级用户才能执行。
/srv: 存放一些服务相关的数据，如网站的文件、数据库等。
/tmp: 临时目录，用于存放临时文件，重启后会被清空。
/usr: 存放用户程序和文件，类似于 Windows 的 Program Files 目录。
/var: 存放经常变化的文件，如日志文件、缓存文件、数据库文件等。
```
